## ✏️백준 10871번 : X보다 작은 수
[문제](https://www.acmicpc.net/problem/10871)

#### 나의 풀이
```swift
let nx = readLine()!.split(separator: " ").map{ Int(String($0))! }
let nums = readLine()!.split(separator: " ").map{ Int(String($0))! }

for n in nums {
    if n < nx[1] {
        print(n, terminator: " ")
    }
}
```
